# 欢迎来到 GoMate

GoMate 是一个极简「地点组队」平台，专为户外爱好者打造。无论你是徒步、登山、露营还是城市探索，都能在这里找到志同道合的伙伴。

## 为什么选择 GoMate？

- **地点为中心** - 基于具体地点组队，精准匹配同行伙伴
- **极简体验** - 无冗余功能，专注于组队核心需求
- **安全可靠** - 实名认证、评价体系、紧急联系人机制
- **社区驱动** - 真实用户分享的路线攻略和最佳实践

## 社区价值观

1. **安全第一** - 永远将人身安全放在首位
2. **环保出行** - 遵循 Leave No Trace 原则
3. **互助友爱** - 团队中的每个人都值得尊重
4. **真实可靠** - 诚信是组队的基础
