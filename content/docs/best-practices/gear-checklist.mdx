# 户外装备清单

合适的装备是户外安全的基础。本指南提供不同难度路线的装备建议，帮助你做好充分准备。

## 装备原则

<Callout type="info" title="装备选择原则">
  1. **适合原则** - 根据路线和环境选择
  2. **轻量原则** - 在满足需求前提下尽量轻便
  3. **可靠原则** - 选择质量可靠的装备
  4. **熟悉原则** - 使用熟悉的装备，不要带新装备走长线
</Callout>

## 基础装备（所有路线必需）

###  footwear

**登山鞋/徒步鞋**
- 防滑大底（Vibram 底最佳）
- 防水透气
- 中高帮保护脚踝
- 提前磨合，不要穿新鞋

<Callout type="error" title="严禁鞋子">
  - 拖鞋、凉鞋
  - 高跟鞋
  - 平底帆布鞋
  - 未磨合的新鞋
</Callout>

### 背包

**一日徒步**：20-30L
**多日徒步**：40-60L

**选择要点**：
- 有腰带分担重量
- 有胸带固定
- 防水或配防雨罩
- 有外挂系统

### 服装

**分层穿衣法**：

| 层次 | 功能 | 材质 | 示例 |
|------|------|------|------|
| 内层 | 排汗 | 速干面料 | 速干 T 恤 |
| 中层 | 保暖 | 抓绒/羽绒 | 抓绒衣 |
| 外层 | 防护 | 防风防水 | 冲锋衣 |

**其他服装**：
- 速干长裤（避免短裤，防刮伤）
- 备用袜子（羊毛或速干）
- 帽子（遮阳/保暖）
- 手套（攀爬/保暖）

### 水和食物

**饮水量参考**：
- 春秋：1.5-2L
- 夏季：2.5-3L
- 冬季：1-1.5L

<Callout type="tip" title="补水建议">
  少量多次，不要等到口渴才喝。建议每 15-20 分钟喝一小口。
</Callout>

**食物选择**：
- 高热量：巧克力、坚果、能量棒
- 易携带：面包、饼干、牛肉干
- 电解质：盐丸、运动饮料
- 备用食物：比预计多带一顿的量

### 导航和通讯

**必备**：
- [ ] 手机（充满电）
- [ ] 充电宝（10000mAh 以上）
- [ ] 离线地图（提前下载）

**建议**：
- [ ] 指北针
- [ ] GPS 设备/手表
- [ ] 对讲机（队伍使用）
- [ ] 哨子

### 照明

**头灯**（比手电更方便）：
- 亮度 200 流明以上
- 续航 8 小时以上
- 防水性能
- 备用电池

<Callout type="warning" title="照明提醒">
  即使计划当日往返，也要携带头灯。时间延误、迷路都可能需要走夜路。
</Callout>

### 急救和防护

**基础急救包**：
```
□ 创可贴（多种尺寸）
□ 无菌纱布
□ 医用胶带
□ 碘伏棉签
□ 弹性绷带
□ 止痛药（布洛芬）
□ 抗过敏药
□ 止泻药
□ 清凉油/风油精
□ 个人药品
```

**防护用品**：
- [ ] 防晒霜（SPF30+）
- [ ] 墨镜
- [ ] 防蚊液
- [ ] 雨衣/冲锋衣

## 按难度分级装备

### 简单路线（如马峦山）

在基础装备上，可以减少：
- 不需要专业登山鞋（运动鞋即可）
- 不需要登山杖
- 不需要头灯（但建议携带）

**可以增加**：
- 溯溪鞋（玩水用）
- 换洗衣物
- 相机

### 中等路线（如七娘山、梧桐山）

**在基础装备上增加**：
- [ ] 登山杖（强烈建议）
- [ ] 护膝（下山用）
- [ ] 更多水（2.5L+）
- [ ] 更多食物
- [ ] 保暖层（山顶风大）

<Callout type="tip" title="登山杖使用">
  正确使用登山杖可以节省 20-30% 体力，减轻膝盖压力。长度调节：手肘 90 度弯曲时手腕与握把平齐。
</Callout>

### 困难路线（如东西冲穿越）

**在基础装备上增加**：
- [ ] 手套（攀爬用）
- [ ] 头盔（防落石）
- [ ] 更多水（3L+）
- [ ] 救生哨
- [ ] 手机防水袋
- [ ] 备用袜子

### 极难路线

**专业装备**（仅限有经验者）：
- [ ] 技术型登山鞋
- [ ] 安全带（需要时）
- [ ] 绳索（需要时）
- [ ] 卫星通讯设备
- [ ] 专业急救包
- [ ] 导航设备（GPS+地图）

## 季节装备

### 夏季装备

<Callout type="warning" title="夏季重点">
  防暑降温是夏季户外的重点！
</Callout>

**增加**：
- 更多水（比平时多 50%）
- 盐丸/电解质补充
- 防晒用品（帽子、墨镜、防晒霜）
- 清凉油/藿香正气水

**减少**：
- 保暖层（但山顶仍需）

### 冬季装备

**增加**：
- 保暖内衣
- 抓绒衣/羽绒服
- 保暖帽
- 保暖手套
- 保温杯（喝热水）

**减少**：
- 水量
- 防晒用品

### 雨季装备

**增加**：
- 雨衣（比雨伞实用）
- 防水袋（保护重要物品）
- 备用衣物
- 防水鞋套

## 装备检查清单

### 出发前检查

```
【服装鞋帽】
□ 登山鞋（检查鞋底磨损）
□ 速干衣裤
□ 保暖层
□ 防风防水层
□ 帽子
□ 手套
□ 备用袜子

【背包物品】
□ 水（足够量）
□ 食物（高热量）
□ 手机（充满电）
□ 充电宝
□ 头灯（检查电池）
□ 急救包
□ 雨具
□ 防晒用品
□ 个人药品

【可选装备】
□ 登山杖
□ 护膝
□ 对讲机
□ 相机
□ 望远镜
□ 坐垫
```

### 装备保养

**每次活动后**：
- 清洁鞋子，晾干
- 清洗衣物
- 检查装备损坏
- 补充消耗品
- 充电电池

**定期维护**：
- 登山鞋：定期清洁、上蜡
- 冲锋衣：定期清洗、恢复防水
- 背包：检查拉链、扣件
- 头灯：检查电池仓

## 装备购买建议

### 预算分配

**入门级（总预算 500-1000 元）**：
- 鞋子：200-400 元
- 背包：100-200 元
- 服装：200-400 元

**进阶级（总预算 2000-5000 元）**：
- 鞋子：500-1000 元
- 背包：300-800 元
- 服装：500-1500 元
- 其他装备：700-1700 元

**专业级（总预算 10000+ 元）**：
- 根据需求配置专业装备

### 品牌推荐

**性价比品牌**：
- 迪卡侬（Decathlon）
- 探路者
- 骆驼

**中端品牌**：
- 凯乐石
- 奥索卡
- 诺诗兰

**高端品牌**：
- 始祖鸟（Arc'teryx）
- 北面（The North Face）
- 巴塔哥尼亚（Patagonia）

<Callout type="tip" title="购买建议">
  新手建议从性价比品牌开始，确定自己的需求后再升级。鞋子最值得投资，建议买好的。
</Callout>
