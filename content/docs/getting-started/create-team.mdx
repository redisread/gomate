# 如何创建队伍

作为领队创建队伍，你可以组织志同道合的伙伴一起探索户外。本指南将帮助你创建一个成功的队伍。

## 创建前的准备

<Callout type="warning" title="领队责任">
  作为领队，你肩负着团队成员的安全责任。请确保自己具备足够的经验和能力。
</Callout>

### 自我评估清单

- [ ] 熟悉该路线的路况和难点
- [ ] 了解当天的天气预报
- [ ] 具备基础的户外急救知识
- [ ] 有处理突发情况的能力
- [ ] 能够控制行进节奏，照顾全队

## 第一步：选择路线

进入路线详情页，点击「创建队伍」按钮。

### 路线选择建议

<Callout type="tip" title="新手领队建议">
  首次带队建议选择自己走过 3 次以上的熟悉路线，难度不宜过高。
</Callout>

## 第二步：填写活动信息

### 基本信息

| 字段 | 说明 | 建议 |
|------|------|------|
| 活动日期 | 出发日期 | 提前 3-7 天发布，给成员准备时间 |
| 集合时间 | 实际出发时间 | 预留 10-15 分钟等人时间 |
| 集合地点 | 具体位置 | 选择交通便利、标识明显的地点 |
| 预计时长 | 全程所需时间 | 按最慢队员估算，宁多勿少 |
| 人数上限 | 最多几人 | 新手建议 6-8 人，便于管理 |

### 集合地点选择

<Cards cols={2}>
  <Card title="推荐地点">
    - 地铁站出口
    - 公交站旁广场
    - 路线入口停车场
    - 有明确标识的地标
  </Card>
  <Card title="避免地点">
    - 私人住宅区
    - 交通不便的位置
    - 信号差的区域
    - 容易迷路的复杂地段
  </Card>
</Cards>

## 第三步：设定参与要求

### 体能要求

根据路线难度设定合理的体能要求：

- **无要求** - 适合休闲路线，如海边栈道
- **能连续步行 3 小时** - 一般徒步路线
- **能连续步行 5 小时** - 中等难度山地
- **有登山经验** - 高难度路线
- **有该路线经验** - 技术型路线

### 装备要求

<Callout type="info" title="必需 vs 建议">
  **必需装备**是参加活动的门槛，**建议装备**供参考。不要设置过高门槛，但安全装备必须强调。
</Callout>

**基础装备（几乎所有路线都需要）：**
- [ ] 运动鞋/登山鞋（禁止平底鞋、拖鞋）
- [ ] 双肩背包
- [ ] 饮用水（建议 1.5-2L）
- [ ] 干粮/路餐
- [ ] 手机+充电宝

**进阶装备（根据路线添加）：**
- [ ] 登山杖
- [ ] 防晒用品（帽子、墨镜、防晒霜）
- [ ] 雨具
- [ ] 头灯/手电筒
- [ ] 急救包

## 第四步：费用说明

### 费用类型

GoMate 平台本身不收取任何费用，但活动可能产生以下费用：

- **交通费** - 包车、拼车费用
- **门票** - 景区入口费用
- **餐饮** - AA 聚餐费用
- **装备租赁** - 如需要统一租赁

### 费用设置建议

<Callout type="tip" title="透明原则">
  费用说明要尽可能详细，避免事后纠纷。建议采用 AA 制，多退少补。
</Callout>

示例：
```
费用预估：
- 包车：人均 30-40 元（按 8 人计算）
- 门票：20 元/人
- 合计：50-60 元/人

集合时收取 60 元，活动结束后多退少补。
```

## 第五步：活动描述

好的活动描述能吸引合适的参与者。

### 描述模板

```
【路线简介】
简要介绍路线特色、景观亮点

【行程安排】
08:00 集合
08:30 出发
12:00 山顶午餐
15:00 到达终点

【注意事项】
- 特别强调的安全事项
- 需要携带的特殊装备
- 其他重要信息

【领队介绍】
简单介绍自己的户外经验
```

## 第六步：发布与审核

填写完成后，点击「发布活动」。

### 发布后流程

<Steps>
  <Step step={1} title="等待申请">
    成员开始提交加入申请
  </Step>
  <Step step={2} title="审核成员">
    查看申请者的资料和经验，决定是否通过
  </Step>
  <Step step={3} title="确认名单">
    人数达到预期后，可以关闭申请
  </Step>
  <Step step={4} title="建群沟通">
    通过 GoMate 群聊统一通知细节
  </Step>
</Steps>

## 第七步：审核成员

### 审核标准

<Cards cols={2}>
  <Card title="优先考虑">
    - 资料完整，有真实头像
    - 经验描述详细诚实
    - 装备准备齐全
    - 评价记录良好
  </Card>
  <Card title="谨慎考虑">
    - 资料空白或虚假
    - 明显超出能力范围
    - 装备严重不足
    - 有负面评价记录
  </Card>
</Cards>

<Callout type="warning" title="宁缺毋滥">
  宁可人数少一点，也不要让不合适的成员加入。安全永远是第一位的。
</Callout>

## 第八步：活动管理

### 出发前 1-2 天

- [ ] 在群里再次确认集合时间和地点
- [ ] 提醒天气情况和装备准备
- [ ] 确认最终人数，安排交通
- [ ] 分享路线轨迹文件（如有）

### 活动当天

- [ ] 提前到达集合点
- [ ] 点名签到，核对装备
- [ ] 简要说明安全事项和行程
- [ ] 统一出发，控制节奏

### 行进中

- [ ] 保持队伍紧凑，前后呼应
- [ ] 定时休息，关注队员状态
- [ ] 危险路段提醒和协助
- [ ] 拍照留念，记录美好时刻

## 领队最佳实践

<Callout type="success" title="优秀领队的特质">
  - **责任心** - 始终把队员安全放在首位
  - **沟通能力** - 清晰表达，及时反馈
  - **应变能力** - 能处理突发状况
  - **包容心** - 理解不同水平队员的需求
  - **环保意识** - 带头践行 Leave No Trace
</Callout>

### 建议的领队行为

1. **提前踩线** - 对不熟悉的路线提前探路
2. **准备预案** - 准备下撤路线和紧急联系方式
3. **携带急救包** - 基础药品和急救用品
4. **控制节奏** - 按最慢队员的速度前进
5. **及时止损** - 天气恶化或队员不适时果断下撤

## 紧急情况处理

如遇紧急情况，按以下流程处理：

<Steps>
  <Step step={1} title="评估状况">
    判断伤情/险情的严重程度
  </Step>
  <Step step={2} title="确保现场安全">
    将伤者转移到安全位置
  </Step>
  <Step step={3} title="实施急救">
    在能力范围内进行急救处理
  </Step>
  <Step step={4} title="联系救援">
    必要时拨打 110/120 或联系景区救援
  </Step>
  <Step step={5} title="通知平台">
    事后向 GoMate 报告事件经过
  </Step>
</Steps>
