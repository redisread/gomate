# 如何加入队伍

加入已有的队伍是最快开始户外之旅的方式。本指南将带你完成从浏览到加入的完整流程。

## 第一步：浏览路线

在 GoMate 首页或路线库中，你可以看到深圳及周边的精选路线。

<Callout type="tip" title="筛选技巧">
  使用筛选器按难度、时长、景观类型查找适合自己的路线。新手建议从「简单」或「中等」难度开始。
</Callout>

### 路线信息解读

每条路线卡片包含以下信息：

- **难度等级** - 简单 / 中等 / 困难 / 极难
- **预计时长** - 正常行进所需时间
- **累计爬升** - 海拔变化总量
- **景观类型** - 山海 / 森林 / 城市 / 海岸线
- **最佳季节** - 推荐前往的月份

## 第二步：查看队伍

进入路线详情页后，点击「当前队伍」标签，查看该路线的实时组队信息。

### 队伍卡片信息

<Cards cols={2}>
  <Card title="基本信息">
    出发时间、集合地点、预计结束时间
  </Card>
  <Card title="人数状态">
    当前人数 / 人数上限、申请中人数
  </Card>
  <Card title="领队信息">
    领队昵称、经验等级、历史评价
  </Card>
  <Card title="活动要求">
    装备要求、体能要求、费用说明
  </Card>
</Cards>

## 第三步：申请加入

找到合适的队伍后，点击「申请加入」按钮。

### 填写申请信息

<Steps>
  <Step step={1} title="自我介绍">
    简要介绍自己的户外经验、体能状况。如果是新手，诚实说明，大多数领队都欢迎有热情的新手。
  </Step>
  <Step step={2} title="确认装备">
    勾选已准备的装备清单，或说明暂时缺少的装备。
  </Step>
  <Step step={3} title="联系方式">
    确认手机号（仅对领队可见），用于紧急联系。
  </Step>
  <Step step={4} title="提交申请">
    提交后等待领队审核，通常会在 24 小时内回复。
  </Step>
</Steps>

<Callout type="warning" title="注意事项">
  请如实填写信息。虚假信息可能导致申请被拒，或在活动中造成安全隐患。
</Callout>

## 第四步：等待审核

提交申请后，你可以在「我的申请」中查看状态。

### 申请状态说明

- **审核中** - 领队尚未处理，请耐心等待
- **已通过** - 恭喜你！可以加入群聊，准备出发
- **已拒绝** - 查看拒绝原因，可以申请其他队伍
- **已取消** - 你主动取消的申请

<Callout type="info" title="提高通过率的小技巧">
  - 完善个人资料，添加真实头像
  - 详细描述相关经验
  - 提前准备好必需装备
  - 选择与自己水平匹配的队伍
</Callout>

## 第五步：确认参加

申请通过后，你需要在 24 小时内确认参加。

### 确认前请检查

- [ ] 时间安排无冲突
- [ ] 已准备好必需装备
- [ ] 了解集合地点和交通方式
- [ ] 告知家人/朋友出行计划

<Callout type="error" title="重要提醒">
  确认参加后请遵守承诺。临时放鸽子会影响你的信用评分，多次违约可能被限制使用平台。
</Callout>

## 第六步：加入群聊

确认参加后，你将自动加入该活动的群聊。

### 群聊使用指南

- **自我介绍** - 进群后简单介绍自己
- **关注公告** - 领队会发布重要通知
- **及时沟通** - 如有变动请第一时间告知
- **尊重他人** - 保持友善，避免无关话题

## 第七步：活动当天

### 出发前检查清单

- [ ] 查看天气，准备相应装备
- [ ] 充足睡眠，提前吃好早餐
- [ ] 充满手机电量，携带充电宝
- [ ] 携带身份证件（部分路线需要）
- [ ] 告知家人具体行程

### 集合流程

1. 提前 10-15 分钟到达集合点
2. 寻找领队或队伍标识
3. 签到确认，核对装备
4. 统一出发，保持队形

## 第八步：活动评价

活动结束后，请对领队和队友进行评价。

### 评价维度

- **准时性** - 是否按时到达集合点
- **团队协作** - 是否愿意帮助他人
- **环保意识** - 是否遵守环保原则
- **整体体验** - 综合满意度

<Callout type="success" title="评价的重要性">
  真实的评价帮助建立社区信任体系，让优秀的领队和参与者脱颖而出。
</Callout>

## 常见问题

**Q: 申请被拒绝怎么办？**
A: 查看拒绝原因，可能是人数已满、装备不符或经验要求不匹配。可以申请其他队伍或提升自己的准备。

**Q: 可以带朋友一起参加吗？**
A: 需要分别提交申请，或在申请时注明同行人数，等待领队确认。

**Q: 活动取消怎么办？**
A: 如遇恶劣天气或其他不可抗力，领队可能取消活动。费用将原路退回，你也可以申请加入其他队伍。
